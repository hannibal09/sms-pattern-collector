How to Contribute

Step 1: Get SMS Backup

We recommend using the popular SMS Backup & Restore app on Android.

  1. Install SMS Backup & Restore from Play Store.
  2. Open app and tap Set up a backup.
  3. Select Messages only.
  4. Important: You can tap "Advanced Options" to select only specific conversations if you prefer, but our tool filters them anyway.
  5. Choose Local Backup (store on phone).
  6. Run the backup. It will create an `.xml` file.

Step 2: Clean & Anonymize

  1. Transfer the XML file to your computer (or open this site on mobile).
  2. Drag and drop the file into the main page.
  3. Wait for processing to complete.

Step 3: Review & Share

  1. Scroll through the Preview section.
  2. We show you exactly what changed. Green text is the anonymized version we keep.
  3. If you see any message you don't like, click the Trash Icon to remove it.
  4. Click Download JSON.
  5. Email or share the JSON file with the developer.
